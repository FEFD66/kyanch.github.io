更新系统时钟

磁盘分区、格式化、挂载

如果你的硬盘已有系统，并且想保留原有的系统，请先阅读多系统引导部分
按照自己的想法分区就好。
注意4k对齐和特殊功能分区(ESP分区)的格式(fat32)
磁盘分区使用 fdisk或parted

格式化 命令
extX(X=2/3/4) mkfs.extX
fat32 mkfs -t vfat
swap mkswap(启用:swapon/关闭:swapoff)

如果磁盘紧张的话可以参照这个，但是长期使用可能导致磁盘不足，如果只是学习用、虚拟机用则完全可以:

挂载点 分区 用途 格式化类型 大小
/mnt/boot /dev/sdX1 EFI启动分区 FAT32 260MB
/mnt /dev/sdX2 根目录分区 EXT4 20GB

安装

选择镜像

修改镜像列表文件

直接修改iso内的源，安装时会自动覆盖

这里使用南京大学源

镜像列表有非常多的镜像站点，但具体到个人使用时，建议只保留最快的一个到两个。

使用reflector可以筛选出最快的镜像

可以通过rankmirrors工具筛选出最快的镜像

liveCD联网后会自动使用reflector选出最快的70个镜像。如果在其他环境则需要安装pacman-contrib 包

更多细节参见: Mirrors

安装必要的包

注意：这里安装的包都是官方源中的包，也就是说你可以之后用pacman安装这里的可选包，也可以将自己要用的包加入这个命令中。

事实上： pacstrap只是把软件包安装到另外一个目录中

如果你不想看这部分下面的文字，直接运行这个命令

其中 /mnt是根分区挂载的路径
base 是必须的组包
另外内核包也必须有至少一个:

  • linux 是内核包
  • linux-lts 是长期支持内核包

考虑稳定性 可以使用长期支持版本的内核包(linux-lts)，追求新版本可以用linux包。

linux-firmware 是驱动包 。如果使用WiFi作为主要的网络链接方式，则强烈建议安装此包，除非你确定内核对你的网卡有相应的驱动。
base-devel 是基本开发工具包组，可以不装。包含的工具可以查看这里

理论上你可以在此写上任何源里有的包，但是考虑安装的速度，先把基本系统安装之后，再根据需要使用包管理器安装。
